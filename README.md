# I-Nelory

I-Nelory is a personal digital memory platform.

This repository is prepared as a simple full-stack TypeScript workspace:

- `frontend` - React, Vite, and TypeScript
- `backend` - Node.js, Express, and TypeScript

Firebase, Cloudinary, Gemini, authentication, database models, and API endpoints are intentionally not implemented yet.

## Scripts

From the project root:

```bash
npm run dev:frontend
npm run build:frontend
npm run dev:backend
npm run build:backend
```

You can also run commands inside each workspace directly:

```bash
cd frontend
npm run dev
```

```bash
cd backend
npm run dev
```
