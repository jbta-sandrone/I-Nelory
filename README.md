## рџ“– About

I-Nelory is a full-stack web application that serves as a private digital memory journal. Unlike traditional social media platforms, I-Nelory focuses on preserving personal memories rather than sharing posts publicly.

Users can upload photos and videos, organize them into albums, search memories using AI, track storage usage, customize their experience, and safely manage their personal archive.
